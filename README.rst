aspects-unidigital plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/eduNEXT/tutor-contrib-aspects-unidigital

Usage
-----

::

    tutor plugins enable aspects-unidigital
